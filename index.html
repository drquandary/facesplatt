Methodology

Each portrait from the Chicago Face Database is reconstructed as a 3D Gaussian Splat using FaceLift (Lyu et al., ICCV 2025): multi-view diffusion synthesises 6 novel views from the single input image, then a GS-LRM transformer predicts ~500k Gaussian primitives with position, opacity, scale, rotation, and spherical-harmonic colour.

Inference was run on a single NVIDIA B200 GPU on UPenn's PARCC Betty cluster. End-to-end pipeline (2D portrait → 3D splat) averages ~30 s per subject. Click any face above to load its splat in an interactive 3D viewer.
